# Support

EU5Miner is a public preview project, so support is best effort.

## Getting Help

- Open a GitHub issue for reproducible bugs or concrete feature requests.
- Include your `eu5miner` version, Python version, OS, and exact command or code sample. If the issue is specific to MCP or GUI, also note the relevant package version (`eu5miner-mcp` or `eu5miner-gui`).
- State whether the problem requires a local EU5 install and, if so, which workflow or file family is involved.
- Keep repro cases as small and install-independent as possible.

## Security

Do not report security issues in public issues.

See [SECURITY.md](SECURITY.md) for the current vulnerability reporting process.
