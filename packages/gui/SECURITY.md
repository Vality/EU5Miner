# Security Policy

## Supported Versions

This repository is currently published as a public preview.

| Version | Supported |
| --- | --- |
| 0.5.x | Yes |
| Older preview versions | No |

## Reporting a Vulnerability

Please report suspected vulnerabilities privately.

- Do not open a public issue, pull request, or discussion with exploit details.
- Use GitHub's private vulnerability reporting for this repository if it is available.
- If private reporting is not available, contact the maintainer through the contact details on the GitHub profile and include the repository name, affected version, reproduction steps, impact, and any suggested mitigation.

The goal for this small preview repo is to acknowledge reports promptly, confirm severity, and publish a fix or mitigation as soon as practical.

## Scope

- This policy covers the source code and packaged artifacts published from this repository.
- It does not cover vulnerabilities in a local game installation, third-party mods, or the game itself.
