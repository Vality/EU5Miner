## 0.6.0 - 2026-04-06

- Added the structured read-only browser session used by the GUI shell, including install overview, stable inspection-backed report pages, and covered entity list/detail browsing.
- Added direct page selection and browser ergonomics such as page aliases, page-index listing and filtering, page windowing, session-summary context, navigation hints, and CLI reopen hints.
- Added large-entity browsing controls for sort order, list windows, compact versus detail row rendering, and list-to-detail jump guidance.
- Added thin grouped-package helper pages: diplomacy `war-flow` and `diplomacy-graph`, plus religion `religion-overview`, all built from representative install files and stable `eu5miner.domains` helpers.
- Aligned preview release metadata and documentation for the 0.6.0 cut, including pinning the published `eu5miner` dependency to the coordinated core `v0.6.0` release tag.
