# Support

## Getting Help

- Check `README.md` for the current preview scope and command usage.
- Check `documents/specs/README.md` and `documents/architecture.md` for project direction.

## Where To Report Things

- Bug reports: open a GitHub issue with the bug report template.
- Feature requests: open a GitHub issue with the feature request template.
- Security issues: follow `SECURITY.md` and do not open a public issue.

If you are unsure whether a problem belongs in this repo or the core `eu5miner` library, open an issue here with the details and the relevant command or stack trace.
